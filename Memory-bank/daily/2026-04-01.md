## Work Summary
- **Repo image replace**: Users can now replace images in their GitHub repo directly from Browse Assets. Opens a modal with side-by-side current/new preview, reads file as base64, commits the new version via GitHub API.
- **Image dimensions**: Auto-detected via naturalWidth/naturalHeight on load, displayed on cards and in View modal.
- **`putRepoFileBinary()`**: New github.js function for uploading binary files (images) with raw base64 content.
- **Video upload support**: Upload tab now auto-detects video files (mp4/webm), skips image compression, preserves original format. Worker stores with correct video MIME type. Both Upload and Browse Assets render videos with `<video>` player.
- **R2 public access issue diagnosed**: The r2.dev subdomain returns Unauthorized — must be enabled per-bucket in Cloudflare Dashboard.
